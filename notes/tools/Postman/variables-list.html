# Postman 动态变量

下面是 Postman 动态变量列表，它们的值是在请求/集合运行期间随机生成的。

若要在请求前或测试脚本中使用动态变量，你需要使用 pm.variables.replaceIn(),

e.g. pm.variables.replaceIn('{{$randomFirstName}}')。
